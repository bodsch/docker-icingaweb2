# supported Environment Vars

for MySQL Support:
  - MYSQL_HOST
  - MYSQL_PORT
  - MYSQL_ROOT_USER
  - MYSQL_ROOT_PASS
  - IDO_PASSWORD

Authentication
  - ICINGAWEB_ADMIN_USER
  - ICINGAWEB_ADMIN_PASS

for Livestatus Support:
  - LIVESTATUS_HOST
  - LIVESTATUS_PORT
